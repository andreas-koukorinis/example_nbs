johansen(X, rank, model=0, significance_level=1)
Identify cointegating bases of a multivariate I(1) time series using the
Johansen test.
 
The method allows for n > 1 cointegrating relationships; the rank argument
controls the number of bases to be returned.
 
:param X: A multivariate time-series (regularly spaced) and size (N*M),
    where N is the number of observations and M is the number of time
    series.
:param rank: The cointegration basis rank (effectively the number of
    relationships to extract) which should be r <= M.
:param model: Regression model used in performing the trace test (see
    sgmresearchbase.coint.johansen.Johansen).
:return: The cointegration bases (r*M).
normalise(b)
Apply L2 normalisation to a vector.
 
The L2 norm, |x|, of a (complex) vector, x, is defined by the square root
of the sum of the squared values of x: sqrt(sum_i x_i^2). The normalised
vector is then given by x/|x|.
 
:param b: A given basis vector.
:return: A normalised basis vector.
space_dissimilarity(M1, M2, eps=1e-12)
Compute the dissimilarity between two matrices of the same dimensions using
the Grassman manifold distance between column spaces.
 
:param M1: Any real matrix of dimensions M*N.
:param M2: Any real matrix of dimensions M*N.
:param eps: Tolerance to be used in the null space computation.
:return: Distance metric.
spectral(X, rank, num_lags=2)
Return the estimated bases of the multivariate time series (X), assuming a
known rank (rank), lead-lag relationship (num lags) and ADF test for
unit-root testing.
 
This method takes an eigenvector-based approach.
 
:param X: A multivariate time-series (regularly spaced) and size (N*M),
    where N is the number of observations and M is the number of time
    series.
:param rank: The cointegration basis rank (effectively the number of
    relationships to extract) which should be r <= M.
:param num_lags: Number of lead-lag terms used to compute the
    autocovariance matrix W.
:return: The cointegration bases (r*M).
spectral_autolag(X, rank, max_lags=10)
Estimate the cointegrating bases for a multivariate time series using the
spectral method, but also estimating the number of lead-lag terms used to
obtain the autocovariance matrix W (upper bounded by the argument max
lags).
 
:param X: A multivariate time-series (regularly spaced) and size (N*M),
    where N is the number of observations and M is the number of time
    series.
:param rank: The cointegration basis rank (effectively the number of
    relationships to extract) which should be r <= M.
:param max_lags: Maximum number of lead-lag terms to consider in
    computation of the autocovariance matrix W.
:return: The cointegration bases (r*M).
