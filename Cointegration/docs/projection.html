interpolate(index, values, order=3, smoothing=1, resolution=1)
Interpolate irregular time series and apply a univariate spline.
 
The process is as follows:
    1) Compute a new, regular index for the time series based on a given
       resolution and the min/max values.
    2) Apply a 1d interpolation to the time series to obtain a complete,
       regular time series.
    3) Apply a univariate spline to the time series with user-chosen
       arguments for the order and smoothing.
 
:param index: Vector of times (floats) for the irregular time series.
:param values: Vector of values associated with times in `index`.
:param order: The order for the polynomial spline to use.
:param smoothing: Smoothing factor to use with the spline.
:param resolution: Interval between points for the new regular time series.
:return: A regular, interpolated and smoothed time series.
project(X, space=None)
Project multivariate time series onto cointegration bases.
 
Projects an N*M multivariate time series into the cointegration space given
by M*R cointegration basis vector; if no basis is provided (space argument)
then one will be computed using the Johansen method assuming rank 1. Here N
refers to the number of observations, M to the number of markets (time
series) and R is the number of cointegrating bases.
 
:param X: Multivariate time series where rows are observations.
:param space: Cointegration basis.
:return: Matrix (N*R) of spread series (R >= 1).
rescale(ts, limits=(0, 1))
Normalise M, N-dimensional time series to the limits provided.
 
Each of the M time series are normalised using the max and min, indpendent
of each other. This operation is merely applied along each column
one-by-one.
 
:param ts: An (N*M) time series where rows are observations.
:return: Rescaled matrix (N*M).
