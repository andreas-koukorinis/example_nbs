empirical(X, num_lags=2)
Estimates the number of stationary projections of the multivariate I(1)
time series, given an assumed lead lag relationship (num lags), by
identification of unit roots using the ADF test.
 
:param X: Multivariate (M-dimensional) time series of length T.
:param num_lags: Number of lead-lag terms used to compute the
    autocovariance matrix W.
:return: Cointegration rank estimate.
johansen(X, model=0)
Identify if any cointegrating relationships exist between multivariate I(1)
time series and return the estimated cointegration rank using the Johansen
trace test.
 
:param X: Multivariate (M-dimensional) time series of length T.
:param model: Regression model used in performing the trace test (see
    sgmresearchbase.coint.johansen.Johansen).
:return: Cointegration rank estimate.
