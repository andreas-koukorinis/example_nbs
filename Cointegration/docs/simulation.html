Generate a cointegrating, multivariate time series (ECM model) from the
alpha and beta matrices using the method described in Sugita ("Bayesian
Analysis of Dynamic Multivariate Models with Multiple Structural Breaks").
 
Note that alpha and beta must satisfy the stability condition abs(rho(I +
np.dot(beta.T,alpha))) < 1, where rho is the spectral radius.
